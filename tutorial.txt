1. Tutorial de uso do algoritmo de Kruskal (mst.py e codigobase.py)

O arquivo `mst.py` contém a implementação do algoritmo de Kruskal para encontrar a Árvore Geradora Mínima (Minimum Spanning Tree - MST) e a função para visualizá-la:
- kruskal(G)
- visualizar_grafo_arvore(G, arvore, ponderado)

2. Executando o algoritmo de Kruskal

Para este algoritmo, sugerimos utilizar os grafos de exemplo fornecidos nos arquivos `grafo08.txt` e `grafo09.txt`. 

Assim como nos outros algoritmos, sempre antes de rodar o código, é necessário mudar o nome do arquivo do grafo de entrada que será lido[cite: 2]. A execução principal deve ser feita através do arquivo `codigobase.py`.

- Exemplo de definição do grafo no arquivo principal:
    nome_arquivo = "arquivos/grafo08.txt" # ou altere para "arquivos/grafo09.txt"
    file = os.path.join(system_dir, nome_arquivo)
    G, ponderado = ler_grafo_arquivo(file)

3. Executa Kruskal

Após a leitura do grafo, basta chamar a função passando o grafo `G` como parâmetro. O algoritmo retornará um novo subgrafo contendo apenas as arestas da árvore geradora mínima.

# Encontrar a árvore geradora mínima
arvore_geradora_minima = mst.kruskal(G)

4. Como são os prints de saída (Resultados)

Para ver o resultado, você deve iterar sobre as arestas da árvore geradora mínima gerada, somando os pesos de cada aresta.

- Exemplo de código para a saída:
    peso_total = 0
    print("\n Arvore minima: ")
    for u, v, data in arvore_geradora_minima.edges(data=True):
        print(f"{u} - {v} (peso: {data['weight']})")
        peso_total += data['weight']
    
    print(f"Peso total da árvore geradora mínima: {peso_total}")

- Exemplo de saída no terminal:
    Arvore minima: 
    A - D (peso: 5.0)
    C - E (peso: 5.0)
    ...
    Peso total da árvore geradora mínima: 39.0

5. Visualização do Grafo

Para gerar a interface gráfica mostrando a árvore final, o código utiliza a função `visualizar_grafo_arvore`. Ela pintará as arestas da árvore geradora mínima de azul (mais grossas) e o restante das arestas ignoradas de cinza.

- Exemplo de execução:
    mst.visualizar_grafo_arvore(G, arvore_geradora_minima, ponderado)
    plt.show()
